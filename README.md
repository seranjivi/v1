# TalentOS - Recruitment Management System

A modern, interactive talent acquisition and recruitment management dashboard built with React.

## Features

- **Dashboard Overview**: Real-time recruitment metrics and analytics
- **Candidate Management**: Track candidates through the hiring pipeline
- **Job Postings**: Manage active job listings and applications
- **Interview Scheduling**: Schedule and track interview progress
- **Interactive Pipeline**: Drag-and-drop candidate stage management
- **Analytics Dashboard**: Recruitment insights and reporting
- **AI Screening**: Automated candidate evaluation tools

## Tech Stack

- **React 19** - Frontend framework
- **Vite** - Build tool and development server
- **JavaScript/JSX** - Language and syntax
- **CSS-in-JS** - Styling approach

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open your browser to `http://localhost:3001`
